# ziVPN installation

```
wget -O install.sh https://raw.githubusercontent.com/Rhyuu11/zivpn/main/install.sh; sudo chmod +x install.sh; sudo ./install.sh
```

# ziVPN uninstall

```
sudo wget -O uninstall.sh https://raw.githubusercontent.com/Rhyuu11/zivpn/main/uninstall.sh; sudo chmod +x uninstall.sh; sudo ./uninstall.sh
```
